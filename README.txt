CHMPane is a Java program to process compiled html (.chm) files. It has a stream 
based API for accessing CHM file content and provides a GUI to navigate and view 
the content.

To launch the program, just double click the .jar file, 
or run it with "javaw.exe -jar chmpane-{version}.jar".

The latest version can be found at http://chmpane.sourceforge.net 
or http://sourceforge.net/projects/chmpane.
